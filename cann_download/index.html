CANN快速安装

CANN快速安装提供最新特性的CANN社区版本，供开发者提前下载使用。若您期望使用商用标准的CANN版本，请点击商用版获取。

安装指南

📋
环境要求

适用于 CentOS 7+/8+ / openEuler 20.03+/22.03+，需 root 或 sudo 权限

1 安装 shadow 工具包 新增步骤

部分最小化安装的系统缺少 groupadd 等用户管理命令，需先安装 shadow 包。

# 安装 shadow 工具包 (提供 groupadd / useradd 等命令) sudo yum install -y shadow-utils
2 配置用户属组

安装 CANN 前需创建运行所需的用户和用户组

# 创建用户组 sudo groupadd HwHiAiUser # 创建用户并加入用户组 sudo useradd -g HwHiAiUser -d /home/HwHiAiUser -m HwHiAiUser # 设置用户密码 sudo passwd HwHiAiUser
3 安装依赖&配源

安装所需依赖包并配置 CANN 官方软件源

# 安装依赖包 sudo yum install -y wget curl openssl-devel bzip2-devel libffi-devel # 配置 CANN Yum 源 sudo tee /etc/yum.repos.d/ascend.repo << EOF [ascend] name=Ascend Repository baseurl=https://repo.huawei.com/ascend/cann/latest/ enabled=1 gpgcheck=0 EOF
4 安装 NPU 驱动

安装 Ascend 芯片的 NPU 驱动程序

# 安装 NPU 驱动 sudo yum install -y ascend-driver
5 安装 Toolkit

安装 CANN Toolkit 开发套件

# 安装 CANN Toolkit sudo yum install -y ascend-cann-toolkit # 配置环境变量 source /usr/local/Ascend/ascend-toolkit/set_env.sh
6 安装后验证

验证 CANN 安装是否成功

# 检查驱动版本 npu-smi info # 检查 Toolkit 版本 atc --version # 运行示例程序验证 cd /usr/local/Ascend/ascend-toolkit/latest/samples python3 sample_resnet.py

环境要求

需已安装 Anaconda/Miniconda，建议创建独立环境

⚠️ Python版本要求：CANN社区版支持的Python版本为3.7-3.10，推荐使用Python 3.8以获得最佳兼容性。
1 创建指定Python版本的Conda环境

创建独立的Conda环境，指定Python版本以便兼容CANN

# 创建Conda环境（Python 3.8） conda create -n ascend python=3.8 -y # 激活环境 conda activate ascend
2 配置Conda Channel

添加CANN官方Conda源并配置channel优先级

# 添加Ascend conda源 conda config --add channels https://repo.huawei.com/ascend/conda # 设置channel优先级 conda config --set channel_priority strict
3 安装Toolkit

安装CANN Toolkit开发套件

# 安装CANN Toolkit conda install -y ascend-cann-toolkit
4 配置环境变量

配置CANN环境变量，使其在终端中可用

# 配置环境变量 source /usr/local/Ascend/ascend-toolkit/set_env.sh # 验证安装 atc --version

CANN 安装指南

本文档详细介绍了如何在不同操作系统和硬件平台上安装和配置CANN。

概述 安装前提 安装步骤 配置说明 安装验证 常见问题

概述

CANN（Compute Architecture for Neural Networks）是华为公司自主研发的AI异构计算架构，为深度学习训练和推理提供强大的计算能力支持。

主要特性

  • 支持多种深度学习框架（TensorFlow、PyTorch、MindSpore等）
  • 高效的算子优化和并行计算能力
  • 完善的工具链支持，包括编译器、调试器、性能分析工具
  • 支持多种硬件平台和操作系统

产品架构

CANN主要包含以下核心组件：

  • NPU驱动：负责与硬件设备通信，提供基础计算能力
  • Toolkit：包含算子库、编译器、运行时等核心组件
  • Framework Adapter：提供与主流深度学习框架的接口适配
  • 运行时库：提供模型推理和训练的运行时环境

安装步骤

准备工作

  1. 检查系统版本：cat /etc/os-release
  2. 检查CPU架构：uname -m
  3. 检查内存：free -h
  4. 检查磁盘空间：df -h

在线安装（Yum方式）

# 1. 安装依赖包
yum install -y shadow-utils wget curl

# 2. 创建用户组和用户
groupadd ascend && useradd -g ascend ascend
usermod -aG wheel ascend

# 3. 配置CANN软件源
wget https://ascend-repo.huawei.com/CANN/8.0.RC3/CANN-community.repo -O /etc/yum.repos.d/CANN-community.repo

# 4. 安装NPU驱动
yum install -y ascend-driver
systemctl start ascend-driver

# 5. 安装Toolkit
yum install -y cann-toolkit

# 6. 加载环境变量
source /usr/local/Ascend/ascend-toolkit/set_env.sh

# 7. 验证安装
ascend-dmi

离线安装（推荐内网环境）

# 1. 下载离线安装包
# 访问 https://www.huaweicloud.com/intl/zh-cn/ascend/download.html
# 下载对应版本的离线包：Ascend-cann-toolkit_8.0.RC3_linux-x86_64.run

# 2. 上传并解压
chmod +x Ascend-cann-toolkit_8.0.RC3_linux-x86_64.run
./Ascend-cann-toolkit_8.0.RC3_linux-x86_64.run --extract=/tmp/cann_install

# 3. 运行安装脚本
cd /tmp/cann_install
./install.sh --install-path=/usr/local/Ascend

# 4. 加载环境变量
source /usr/local/Ascend/ascend-toolkit/set_env.sh

配置说明

环境变量配置

# CANN环境变量
export ASCEND_HOME=/usr/local/Ascend
export PATH=$ASCEND_HOME/ascend-toolkit/latest/bin:$PATH
export LD_LIBRARY_PATH=$ASCEND_HOME/ascend-toolkit/latest/lib64:$LD_LIBRARY_PATH
export PYTHONPATH=$ASCEND_HOME/ascend-toolkit/latest/python/site-packages:$PYTHONPATH

建议将以上配置添加到 ~/.bashrc 或 /etc/profile 中，以便每次登录时自动加载。

常见问题

安装失败

问题：依赖包冲突
解决方案：使用yum update更新系统后重试，或使用离线安装方式。
问题：网络连接失败
解决方案：检查网络连接，配置代理服务器，或使用离线安装包。
问题：磁盘空间不足
解决方案：清理磁盘空间，确保/usr/local目录有至少10GB可用空间。
问题：权限不足
解决方案：使用sudo或以root用户运行安装命令。

运行时错误

问题：设备权限不足
解决方案：将用户添加到video组和ascend组，重新登录后重试。
问题：环境变量未配置
解决方案：执行source /usr/local/Ascend/ascend-toolkit/set_env.sh。
问题：设备忙
解决方案：使用fuser /dev/davinci0查看占用进程，或重启系统。
问题：模型转换失败
解决方案：检查输入数据格式、算子兼容性，或参考官方文档调整转换参数。

性能问题

问题：推理速度慢
解决方案：检查输入数据预处理是否优化，开启算子融合，使用batch推理。
问题：内存不足
解决方案：减小batch size，或使用内存优化的模型版本。
